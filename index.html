§ 01 — The Problem
Every file format created since 1970 shares one fatal flaw: files are deaf, dumb, and blind to their own content.
  • 01 A PDF does not know it is a contract. A JPG does not know it is a medical scan. Remove the application, the file becomes meaningless bytes.
  • 02 Files lose their context over time. Metadata is stripped. Meaning evaporates. A 20-year-old document becomes an archaeological mystery.
  • 03 Every "smart" solution requires an API, an internet connection, or a proprietary platform. The intelligence lives outside the file — not inside.
  • 04 Billions of files are created daily. None of them can answer a single question about themselves. Not one.
§ 02 — The Solution

Introduce
.sema
Format

A new open file format where intelligence is baked into the file itself. No API. No internet. No external application. The file opens in any browser and understands itself.

1
Content
content/original.*
The original file, untouched. Any format. The source of truth preserved forever.
2
Manifest
sema.json
Rich identity card. Who made it, when, why, in what language, with what context.
3
Brain
brain.json
Pre-computed semantic intelligence. Keywords. Summary. Q&A pairs. Entities. Baked in at creation. Zero latency forever.
4
Viewer
view.html
A complete self-contained web app. Opens in any browser. Ask it questions. Get answers. No plugins. No apps.
